# ⚡ 快速使用 Tushare 可转债

## 🎯 3分钟上手

### 1️⃣ 选择数据源
在侧边栏 "📊 数据源" 中选择：
```
Tushare (A股/可转债) ⭐ 推荐
```
*Token 已内置，无需输入*

### 2️⃣ 选择可转债
- **市场类型**：选择 "可转债"
- **可转债代码**：输入 `113050`（南银转债）

### 3️⃣ 设置参数
- **日期范围**：2025-01-01 至今
- **策略**：双均线策略
- **初始资金**：100000

### 4️⃣ 开始回测
点击 "🚀 开始回测" 按钮，等待结果！

## ⚠️ 重要提示

1. **积分要求**：可转债数据需要 **2000积分**
2. **T+0交易**：可转债支持 T+0，但系统暂不支持日内多次交易
3. **数据质量**：Tushare 数据更新及时，质量优秀
4. **退市风险**：部分可转债可能已退市，无法获取数据
