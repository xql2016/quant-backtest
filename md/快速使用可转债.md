# 🚀 快速使用：可转债回测

## ⚠️ 当前限制

**注意：可转债历史数据功能目前受限**

由于 AKShare 依赖库（py_mini_racer）在 Mac 系统上的兼容性问题，暂时无法获取可转债历史K线数据。

**替代方案：**
- ✅ 使用可转债对应的**正股代码**进行回测
- ✅ 例如：国光转债(128039) → 国光股份(002749)
- 📖 详见：[可转债功能说明_当前限制.md](./可转债功能说明_当前限制.md)

---

## 3步开始（使用正股替代）

### 1️⃣ 选择配置
```
数据源：AKShare (A股/港股/可转债)
市场：可转债 ⭐ 新增
```

### 2️⃣ 输入代码
```
可转债代码：128039（国光转债）
回测区间：最近1年
```

### 3️⃣ 选择策略
```
策略：MACD趋势策略
参数：12-26-9（标准参数）
资金：100,000
```

## ⚠️ 重要提示

- ⚠️ 回测按T+1规则（实际可转债是T+0）
- ⚠️ 结果仅供参考，不构成投资建议
- ✅ 选择AA级以上评级更安全
- ✅ 价格<120元风险相对较小
