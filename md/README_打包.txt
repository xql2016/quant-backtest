================================================================================
🪟 在 Windows 上打包
================================================================================

方法1：一键打包（推荐）
-----------------------
双击运行 "windows_build.bat"，脚本会自动：
  ✓ 检查 Python 环境
  ✓ 安装所有依赖
  ✓ 安装 PyInstaller
  ✓ 执行打包

方法2：手动打包
-----------------------
1. 打开命令提示符（CMD）
2. 进入项目目录
3. 运行以下命令：

   pip install -r requirements.txt
   pip install pyinstaller
   pyinstaller build_spec.spec

🔧 常见问题
================================================================================

Q: 打包后文件很大？
A: 这是正常的，包含了所有依赖库。通常 500MB-1GB。

Q: 打包后无法运行？
A: 检查：
   1. Windows 防火墙设置
   2. 杀毒软件是否拦截
   3. 是否有错误提示

Q: 缺少某个模块？
A: 编辑 build_spec.spec，在 hiddenimports 列表中添加缺失的模块。

Q: 如何添加程序图标？
A: 1. 准备 icon.ico 文件
   2. 编辑 build_spec.spec，修改 icon=None 为 icon='icon.ico'
   3. 重新打包

Q: 可以打包成单个 exe 吗？
A: 可以，但启动会慢。使用命令：
   pyinstaller --onefile run_app.py
