# 量化回测工作台 📈

一个功能强大的量化交易回测系统，支持 A股、港股、美股多市场、多策略回测。

## 🌟 在线演示

访问在线版本：[待部署后填写网址]

## ✨ 功能特性

### 🌍 支持多个市场
- 🇨🇳 **A股** - 基于 AkShare 数据源
- 🇭🇰 **港股** - 基于 AkShare 数据源  
- 🇺🇸 **美股** - 基于 yfinance 数据源

### 📊 6种交易策略
1. **MACD趋势策略** - 金叉死叉信号
2. **双均线策略** - 短线长线交叉
3. **RSI超买超卖** - 抄底逃顶策略
4. **布林带突破** - 回归交易策略
5. **波段策略** - 分批建仓+趋势止盈（9个可调参数）
6. **多重底入场策略** - MACD底背离策略

### 📈 专业回测功能
- ✅ 自定义回测时间区间
- ✅ 灵活的策略参数调整
- ✅ 可视化资金曲线和买卖点
- ✅ 详细的交易日志
- ✅ 策略 vs 买入持有基准对比
- ✅ 胜率、收益率等指标统计

## 💡 使用说明

1. **选择市场** - A股/港股/美股
2. **输入股票代码**
   - A股：6位数字（如 600519）
   - 港股：5位数字（如 00700）
   - 美股：股票代码（如 AAPL）
3. **选择策略** - 6种策略任选
4. **调整参数** - 根据需要微调策略参数
5. **开始回测** - 点击按钮，查看结果

## 🎯 波段策略说明

波段策略支持首次和后续波段独立配置：

### 首次建仓参数
- 首次建仓比例：第一次买入的资金比例
- 首次加仓跌幅：跌多少加仓
- 首次止盈涨幅：涨多少止盈
- 首次止盈均线：判断止盈的均线周期

### 后续入场参数
- 突破均线周期：止盈后突破哪个MA重新买入
- 后续建仓比例：重新买入的资金比例
- 后续加仓跌幅：重新买入后跌多少加仓
- 后续止盈涨幅：重新买入后涨多少止盈
- 后续止盈均线：后续止盈判断的均线周期
