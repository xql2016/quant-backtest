问：能否直接在 Windows 上运行？

答：不能直接双击运行 .py 文件！需要先安装 Python 和依赖包。

================================================================================
📊 两种使用方式对比
================================================================================

方式A：打包版本（推荐给普通用户）⭐⭐⭐⭐⭐
─────────────────────────────────────────
✅ 优点：
  • 无需安装 Python
  • 无需安装依赖包
  • 解压即用，双击启动
  • 适合不懂编程的用户

❌ 缺点：
  • 文件较大（约 600MB-1GB）
  • 需要在 Windows 上打包（不能在 Mac 上打包）

使用步骤：
  1. 使用我提供的打包脚本在 Windows 上打包
  2. 分发 dist\量化回测工作台\ 文件夹
  3. 用户双击 量化回测工作台.exe 即可

方式B：源码版本（给技术用户）⭐⭐⭐
─────────────────────────────────────────
✅ 优点：
  • 文件小（几百KB）
  • 可以修改源码
  • 易于更新

❌ 缺点：
  • 需要安装 Python
  • 需要安装依赖包（6个包）
  • 需要一定技术基础

使用步骤：
  1. 安装 Python 3.9+（记得勾选 Add to PATH）
  2. 打开 CMD，cd 到程序目录
  3. 运行：pip install -r requirements.txt
  4. 运行：python run_app.py

在 Windows 电脑上：

🌐 最简单的方式：在线部署（推荐）
================================================================================

不想打包？不想让用户安装？

使用 Streamlit Cloud：
  1. 代码上传到 GitHub（免费）
  2. 在 streamlit.io/cloud 部署（免费）
  3. 获得一个网址，例如：https://your-app.streamlit.app
  4. 分享网址给任何人，无需下载安装

优点：
  ✅ 完全免费
  ✅ 自动更新
  ✅ 跨平台（Windows、Mac、手机都能用）
  ✅ 无需打包

缺点：
  ❌ 需要联网
  ❌ 代码公开（可以设私有仓库但需付费）

💡 技术细节说明
================================================================================

为什么不能直接运行 Python 文件？

1. Python 是解释型语言，需要 Python 解释器
   （就像 Word 文档需要 Word 软件打开）

2. 程序依赖多个第三方库：
   • streamlit - Web 界面框架
   • pandas - 数据处理
   • numpy - 数值计算
   • matplotlib - 图表绘制
   • akshare - A股/港股数据
   • yfinance - 美股数据

3. 这些库都需要单独安装

打包成 exe 后：
  ✓ 包含 Python 解释器
  ✓ 包含所有依赖库
  ✓ 可以直接运行
