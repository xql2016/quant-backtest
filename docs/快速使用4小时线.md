# 🚀 快速使用：4小时线回测

## 3步开始

### 1️⃣ 选择配置
```
数据源：YFinance (全球市场/加密货币)
市场：加密货币
代码：BTC-USD
```

### 2️⃣ 选择时间粒度 ⭐ **新功能**
```
⏰ 时间粒度：4小时线 (4h)
回测区间：最近60天
```

### 3️⃣ 选择策略
```
策略：MACD趋势策略
  - 快线：8
  - 慢线：17
  - 信号：6
```

## 🎯 为什么选4小时？

- ✅ 比日线细致 **6倍**，信号更多
- ✅ 比1小时快 **4倍**，计算高效
- ✅ **波段交易首选**，持仓2-7天
- ✅ 信号质量高，假突破少
