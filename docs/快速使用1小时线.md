# 🚀 快速使用：加密货币1小时线回测

## 5步开始

### 1️⃣ 选择数据源
```
数据源：YFinance (全球市场/加密货币)
```

### 2️⃣ 选择市场
```
市场：加密货币
```

### 3️⃣ 输入代码
```
常用代码：
- BTC-USD  (比特币)
- ETH-USD  (以太坊)
- BNB-USD  (币安币)
```

### 4️⃣ 选择时间粒度 ⭐ **新功能**
```
⏰ 时间粒度：
- 日线 (1d)      ← 中长线策略
- 1小时线 (1h)   ← 短线策略 ⭐
```

### 5️⃣ 设置回测参数
```
回测区间：建议 7-90 天（1小时线）
初始资金：100,000
策略：根据需要选择
```

---

## 📋 推荐配置

### 配置1：比特币短线MACD
```
代码：BTC-USD
时间粒度：1小时线 (1h)
回测区间：最近30天
策略：MACD趋势策略
  - 快线：5
  - 慢线：13
  - 信号：5
```

### 配置2：以太坊RSI
```
代码：ETH-USD
时间粒度：1小时线 (1h)
回测区间：最近60天
策略：RSI超买超卖
  - RSI周期：6
  - 超卖：25
  - 超买：75
```

---

## ⚠️ 重要提示

1. **数据限制**：1小时线最多730天
2. **仅加密货币**：美股、港股、A股只能用日线
3. **参数调整**：1小时线建议使用更小的策略参数
4. **计算时间**：数据量大，回测需要几秒钟
